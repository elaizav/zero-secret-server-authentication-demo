Zero Secret Server Authentication Demo

Невеликий локальний стенд для ілюстрації ідеї автентифікації, у якій сервер не повинен зберігати секрет користувача у відкритому вигляді.

Це навчальний прототип для академічної демонстрації. Він не є готовою продукційною системою автентифікації та не повинен використовуватися для захисту реальних облікових записів.

Локальна симуляція

Введіть ідентифікатор користувача та локальний секрет. Усі обчислення виконуються тільки в браузері за допомогою Web Crypto API.

Що демонструється

  1. Локальне хешування секрету в браузері.
  2. Генерація одноразового виклику `nonce`.
  3. Побудова демонстраційної відповіді на основі ідентифікатора, виклику та похідного значення.
  4. Пояснювальний журнал кроків без мережевих запитів і без збереження даних.

Модель спрощена навмисно: тут немає реєстрації, сервера, сховища, сесій, транспортного захисту, контролю повторних спроб або повноцінної криптографічної схеми перевірки.
