# Zero Secret Server Authentication Demo
Демонстраційний прототип системи автентифікації за моделлю Zero Secret Server Authentication.

## Опис проєкту
Цей проєкт є демонстраційною реалізацією підходів до побудови систем автентифікації без зберігання секретів на сервері. Основна ідея полягає у використанні клієнтського шифрування, stateless-підходу до автентифікації та інструментальних засобів для безпечної роботи з даними користувача.

## Мета
Метою проєкту є дослідження методології та інструментальних засобів розробки систем автентифікації за моделлю Zero Secret Server Authentication і створення демонстраційного прототипу, який ілюструє основні принципи такої архітектури.

## Основні можливості
- автентифікація без зберігання секретів на сервері;
- клієнтське шифрування даних;
- демонстрація stateless-підходу;
- базова структура для подальшого розвитку прототипу;
- підготовка до використання як основи бакалаврської роботи.

## Структура репозиторію
- index.html -основний демонстраційний файл застосунку;
- README.md -опис проєкту;

- .gitignore -перелік файлів і папок, які не потрібно додавати до репозиторію;

## Технології
У межах прототипу можуть використовуватися:
- HTML5;
- JavaScript;
- клієнтські засоби криптографічного захисту;
- сучасні підходи до реалізації безпечної автентифікації.
